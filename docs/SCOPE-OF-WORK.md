# EchoRep MVP Scope (C# / React / Mobile)

## The Technical Core

1. **Backend:** .NET 8 Web API hosted on Railway.
2. **Frontend:** React (Vite) hosted on Vercel.
3. **Mobile:** React Native (Expo) for iOS/Android notifications.
4. **Database:** PostgreSQL (via Railway or Supabase).
5. **CI/CD:** GitHub Actions for automated testing and deployment.

## The MVP Features

- **OAuth2 Flow:** Securely connect Google Business Profile via .NET Identity.
- **Review Hub:** Sync reviews to Postgres; display in React/React Native.
- **AI Engine:** Semantic kernel or direct OpenAI integration in C#.
- **Mobile Alerts:** Push notifications when a 1 or 2-star review hits.
